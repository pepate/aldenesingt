# **App Name**: Aldene Singt

## Core Features:

- PDF Upload: Allow users to upload PDF documents containing lyrics and chords.
- Session Start/Join: Enable users to create and join sessions for synchronized document viewing.
- PDF Display: Display the selected PDF document to all participants in a session.
- Synchronized Scrolling: Synchronize the scrolling of the document for all users in the session, controlled by the host.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a modern and collaborative feel.
- Background color: Light gray (#F5F5F5) to ensure readability of the displayed documents.
- Accent color: Cyan (#00BCD4) for interactive elements and highlights, to create a contrast with the purple primary.
- Body and headline font: 'Inter', a sans-serif for a modern, machined, objective, neutral look that promotes readability.
- Use simple, clean icons for actions like upload, join, and scroll.
- Clean and intuitive layout for easy navigation and document viewing.
- Smooth scrolling transitions to enhance the synchronized viewing experience.
